1 ob-ipython

使用 ob-ipython 可以在 org 文档内运行代码块, 代码块内容会被发送到后台的 Jupyter 进程运行, 返回结果会被插入到 org 文档中. org 文档丰富的显示类型使得使 用 org 文档来代替 jupyter notebook 成为了一种不错的体验.
